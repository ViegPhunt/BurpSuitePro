# BurpSuite Professional
- Created By [t.me/zer0daylab](t.me/zer0daylab) and modified by ViegPhunt.

> [!IMPORTANT]
> **Recommended for use with [JDK 24](https://www.oracle.com/asean/java/technologies/javase/jdk24-archive-downloads.html)**

## Installation on `Windows`
- Link Download JDK 24 for Windows [`here`](https://download.oracle.com/java/24/archive/jdk-24_windows-x64_bin.exe).

> [!NOTE]
> **If an error occurs**, open CMD and run the following commands to clear the preferences:
> ```
> rd /s /q "%userprofile%\AppData\Roaming\BurpSuite\"
> reg delete "HKEY_CURRENT_USER\SOFTWARE\JavaSoft\Prefs\burp" /f
> ```

## Installation on `Linux`
- Link Download JDK 24 for Linux x64 [`here`](https://download.oracle.com/java/24/archive/jdk-24_linux-x64_bin.tar.gz).
